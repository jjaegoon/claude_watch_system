## 함정 ①: precommit staged_secrets FAIL on git-rm'd deletions

**증상**: `git rm memory/episodes/Step1.md Step2.md` 후 precommit-check 실행 시 staged_secrets FAIL.

**원인**: check_staged_secrets가 `git diff --cached --name-only` (삭제 포함 전체 staged 파일 목록)를 순회하며 `git show :$f` 실행. 삭제된 파일은 index에서 제거되어 `git show :$f`가 에러 exit → pipefail → FAIL 판정.

**workaround**: 삭제 파일을 별도 commit으로 분리 (`git restore --staged`로 unstage 후 이번 commit 제외).

**정식 fix (T-39 후보)**: check_staged_secrets에서 삭제 파일 필터링:
```bash
staged="$(git diff --cached --diff-filter=ACMR --name-only)"
```

**참조**: precommit-check.sh check_staged_secrets 함수, gotcha #16과 유사한 메타 함정.

## 해결한 문제

### TSX 파일 제네릭 파싱 충돌

`const fetchJson = async <T>(path: string)` → `<T>`가 JSX 태그로 해석됨.
**처방**: arrow function 대신 `async function fetchJson<T>(path: string)` 함수 선언 사용.

### betterstack.test.ts vi.fn() → undefined.catch 실패

`vi.fn()` 기본값 undefined → `fetch(...).catch(...)` 에서 TypeError.
**처방**: `vi.fn().mockResolvedValue(new Response('', { status: 200 }))` 기본 mock 값 설정.
